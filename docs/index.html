What it is

tokencur reads local AI agent logs (Claude Code, Codex CLI, Kimi Code), values the usage at published API rates as showback — subscriptions don't bill per token — and normalizes it to FOCUS, the FinOps billing standard. Run python -m tokencur report for the terminal summary.
